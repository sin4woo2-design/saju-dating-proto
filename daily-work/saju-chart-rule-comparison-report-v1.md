# [Saju] chart aggregation rule comparison v1

## 비교 대상 규칙안
1. 현행 규칙
   - 천간 1.6 / 지지 1.0
2. 월지 강화 규칙
   - 천간 1.6 / 지지 기본 1.0 + 월지(월주 지지) 2.0
3. 지장간 포함 규칙
   - 현행 규칙 + 지장간 가산(분해비 1: 1.0, 2: 0.7/0.3, 3: 0.7/0.2/0.1, 총가중 0.8)

## S01~S08 strong element 분포 비교
- 현행: wood 2 / fire 0 / earth 4 / metal 0 / water 2
- 월지 강화: wood 1 / fire 0 / earth 4 / metal 1 / water 2
- 지장간 포함: wood 2 / fire 0 / earth 4 / metal 0 / water 2

## 관찰 요약
- 표본 8개 기준에서 earth 강세 편향은 세 규칙 모두 유지됨.
- 월지 강화 규칙만 S06이 earth -> metal로 이동하여 분포 변화가 발생함.
- 지장간 포함 규칙은 현재 표본에서 strong 분포 변화는 없지만, 세부 점수 분산은 변함.

## 편향 가능성 점검
- 현행 규칙은 지장간/월령 미반영으로 구조적 단순화가 큼.
- earth가 4/8로 높게 나오는 점은 branch 매핑 단순화 영향 가능성 있음.
- fire/metal strong이 0건인 점은 표본 특성 + 규칙 단순화가 함께 작용했을 수 있음.

## 추천안
- 다음 단계 추천: **월지 강화 규칙**을 A안으로 채택해 확장 검증
  - 이유: 표본 내에서 실제 분포 변화가 관찰되어 규칙 민감도 확인에 유리
  - 단, 지장간 포함 규칙과 함께 30~50건으로 재검증 필요
