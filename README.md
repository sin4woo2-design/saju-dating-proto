# 시작이 반 (Habit Motivation MVP)

"시작하면 반은 했다"를 핵심 메시지로 만든 캐주얼 습관앱.

## 현재 구현 범위 (결제 제외 상용화 기본기)

- 습관 생성/체크 및 스트릭 관리
- 요일별 활성 습관(오늘 해야 할 습관만 노출)
- 실패 시 **복구 미니게임** + 프리즈 토큰 방어
- 레벨/경험치 시스템(행동 즉시 보상)
- 동기부여 톤 설정 (따뜻한 코치/냉정한 코치/친구 응원)
- 통계/요일별 성공률 분석/성과 배지
- 한 줄 회고(감정 + 메모)
- 커뮤니티 챌린지 참여 + 습관 피드
- 로컬스토리지 기반 데이터 영속화

## 실행

```bash
npm install
npm run dev
```
