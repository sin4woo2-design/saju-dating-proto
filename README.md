# 시작이 반 (Habit Motivation MVP)

"시작하면 반은 했다"를 핵심 메시지로 만든 캐주얼 습관앱 MVP.

## 핵심 기능

- 습관 생성/체크 및 스트릭 관리
- 실패 시 **복구 미니게임**(숫자 맞추기)
- 통계/주간 차트/분석 요약
- 커뮤니티 챌린지 참여 + 다른 사람 습관 피드
- 소액 결제 실험용 프리미엄 탭(가격/상품 구조)

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```
