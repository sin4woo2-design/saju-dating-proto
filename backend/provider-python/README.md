# provider-python (skeleton)

이 디렉터리는 lunar-python 연동 전 단계의 fake provider 서버 설계용 스켈레톤이다.

현재 상태
- 실행 코드 없음
- API 계약 문서 기준으로 폴더 구조만 고정

참고 문서
- `docs/python-provider-api-spec.md`
- `docs/saju-real-provider-spec.md`

예정 구조
- `app/main.py`
- `app/schemas.py`
- `app/routes/saju.py`
- `app/services/fake_engine.py`
- `app/tests/test_saju_routes.py`

주의
- 이번 단계에서는 설치/연동/실행 구현을 하지 않는다.
