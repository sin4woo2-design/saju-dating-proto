# 시작이 반 - 습관앱 벤치마킹 노트

소액 결제 검증 목적의 MVP 기획 참고.

## 벤치마크 대상

1. **Habitica** - 게임화(보상/퀘스트)로 습관 지속성 강화
2. **Fabulous** - 감정 기반 동기 문구 + 루틴 코칭
3. **Done / Streaks** - 매우 간단한 체크 UX와 스트릭 시각화
4. **Forest** - 실패 비용(나무 죽음)으로 집중 행동 유도
5. **Coach.me** - 커뮤니티/코칭 결합형 동기부여

## 공통 패턴

- **초기 진입은 가볍게**: 3분 이내 첫 체크 완료
- **행동 직후 보상**: 애니메이션, 칭찬 문구, 점수
- **실패 완충 장치**: 하루 실수 시 완전 초기화 대신 복구 옵션
- **사회적 압력/응원**: 챌린지, 피드, 친구와 같이 하기
- **유료 전환 포인트**: 분석/리포트/고급 템플릿/코칭 메시지

## 시작이 반 MVP에 반영한 요소

- 실패시 미니게임으로 스트릭 복구 기회 제공
- 습관 템플릿 즉시 추가(진입장벽 축소)
- 통계 + 분석 요약 탭 제공
- 커뮤니티 챌린지 및 타 사용자 습관 피드 제공
- 프리미엄 가격 실험 UI(월 소액 + 1회성 챌린지팩)
